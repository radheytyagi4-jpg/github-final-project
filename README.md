## Simple Interest Calculator

This project contains a Bash script `simple-interest.sh` that calculates **Simple Interest** based on user input.

### Features
- Takes Principal, Rate of Interest, and Time Period as input from user
- Uses standard Simple Interest formula
- Displays the calculated interest clearly

### Formula Used
**Simple Interest = (Principal × Rate × Time) / 100**

### How to Use
```bash
bash simple-interest.sh
